even is a tool for Effective Visualization of Evolving Networks -- [demo](https://ivis-at-bilkent.github.io/even/)

## Software
### Required Tools
- npm
- node.js

### Installation
```
git clone https://github.com/iVis-at-Bilkent/even.git
cd even
npm install
```

### Build and Start
```
npm run build
npm start
```
